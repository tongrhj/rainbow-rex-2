# Rainbow Rex 2
Colour Picker Game with a Dinosaur

* Install the dependencies:

```
yarn
```

* Run development server:

```
yarn start
```

Open the web browser to `http://localhost:8888/`

* Build for production:

```
yarn build
```

### Tools of Choice
1. React
2. Webpack
3. Postcss
4. Standard
